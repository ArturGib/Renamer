
In specified directory (not desktop) find and replace parts of names of objects (folders and files).

It allows to find specified string in names of files 
contained in chosen directory and replace 
all findings to the specified new string.

Example:
directory - 'C://Temp'
files in diretory:
'part_1_old.txt'
'part_2_old.txt'
'part_3_old.txt'
'part_4_old.txt'

How work this app:
1) Specified directory:
'C://Temp'

2)Specified new part of name:

'new'

3)Specified old part of name:

'old'

Output (names in directory):
'part_1_new.txt'
'part_2_new.txt'
'part_3_new.txt'
'part_4_new.txt'
